Planned API Endpoints

GET

/api/v1/words

Retrieve a paginated list of Igala words

GET

/api/v1/words/:slug

Get detailed information about a specific word

GET

/api/v1/search

Search for words by query parameter

GET

/api/v1/examples

Retrieve usage examples in Igala

POST

/api/v1/translate

Translate text between English and Igala

GET

/api/v1/random

Get a random Igala word
